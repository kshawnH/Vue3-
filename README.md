# Vue 3 + TypeScript + Vite

## 下载项目
```shell
git clone git@github.com:youngyangyang04/element-ui.git
```

## 启动项目
```shell
npm install
# 启动文档
npm run docs:dev
# 启动项目
npm run dev
```
